A simple application to manage hotel operations, guest bookings, and room availability.

#Features
Guest Management: Register and maintain guest details.

Room Booking: Check availability and book rooms in real-time.

Check-in/Check-out: Process for guest arrivals and departures.

Billing: Generate invoices for stay and services.

Dashboard: Overview of current occupancy and status.

#Tech Stack
Backend: Python

Database: SQLite

Frontend: Tkinter

#Installation
Clone the repository:
git clone https://github.com/cati3000/Hotel-Management-App.git

Navigate to the directory:
cd Hotel-Management-App

Install dependencies:
pip install -r requirements.txt

#Usage
Run the main script to start the application:
python main.py
